# Data Ingestion Service

Python-based microservice responsible for collecting real-time cryptocurrency market data from various exchanges.

## Overview

This service connects to cryptocurrency exchange APIs to gather:
- Price data (OHLCV)
- Order book data
- Trade data
- Market metrics

## Technology Stack

- Python 3.11+
- FastAPI (planned)
- Async HTTP clients
- Kafka producer
